# VoidLive: React

Exemplo desenvolvido durante a primeira live do VoidNull, no dia 2017-10-30.


## Quick Start

Clone o repositório e:

    cd voidlive-react

    npm install
    npm start

A app ficará acessível em `localhost:8080`.
